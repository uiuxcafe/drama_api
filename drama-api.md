## 1.3 取得熱門列表

_在新聞首頁點選「排序」按鈕時，點選熱門排序，打此 API，即依據一週內新聞按尋夢新聞觀看次數由多至少排序。_ 

- Param
  
    | 參數 | 敘述 | 資料型態 | 必填 |
    | -- | -- | -- | -- | 
    | authtoken| 通行碼 <li>第一次：帶空值</li><li>自動登入：帶入儲存的通行碼</li>| string | Y |
    | otpw| 自動登入時回傳的欄位 | string | Y |

- Example Request
 
```json
{ 
    "authtoken":""
}
```

- Example Response

```json
{ 
    "authtoken":""
}
```
