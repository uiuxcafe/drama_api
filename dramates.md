## 1.4 取得新聞tag列表

_進入新聞詳細頁時，點選下方分頁「相關新聞、相關戲劇、相關討論」任一按鈕時，先打此 api ，取得新聞 tag 列表。_


- query
```
query {
  news_type(where: {news_id:{_eq: "1"}, type: {label: {_eq: "tag"}}}) {
    type {
      name
    }
  }
}

```
- Response
  
```json
{
  "data": {
    "news_type": [
      {
        "type": {
          "name": "陳情令"
        }
      },
      {
        "type": {
          "name": "肖戰"
        }
      },
      {
        "type": {
          "name": "王一博"
        }
      }
    ]
  }
}
```
